# yolo
  Spring MVC Web Project
# 설명
    현재 서비스 하고있는 온라인 음악 스트리밍 사이트를 컨셉으로 하여 만든 YOLO 입니다. Spring MVC 를 기반의 JAVA 웹 기반 개인 프로젝트입니다.
    소스 코드 및 pdf 문서로 작성하였습니다. 프로젝트 설명서를 보시려면 pdf 형식의 문서를 참고해 주시기 바랍니다.
